üìñ Project Overview

The Simple Login Application is a full-stack web application demonstrating modern authentication patterns, role-based access control, and responsive UI design. This project serves as a comprehensive example for learning software engineering principles.

‚öôÔ∏è Backend (Spring Boot)

Key Components

AuthController.java
Handles authentication endpoints: login and profile retrieval
AuthService.java
Business logic for user authentication and JWT token management
JwtService.java
JWT token creation, validation, and extraction utilities
UserDocument.java
MongoDB document model for user data with roles (USER, ADMIN)
UserRepository.java
MongoDB repository for user data operations

Configuration

# application.properties server.port=8080 spring.data.mongodb.uri=mongodb://localhost:27017/simple_login jwt.secret=your-secret-key-here jwt.expiration=86400000

üöÄ Quick Start Guide

Prerequisites

  • Java 21 LTS or higher
  • Node.js 18 or higher
  • Docker (for MongoDB)
  • Git

Setup Steps

  1. Clone the repository:
    git clone https://github.com/belmont-bhagat/substitute-teacher.git
  2. Start MongoDB:
    docker-compose up -d mongodb
  3. Start the backend:
    cd simple-login-backend && ./mvnw spring-boot:run
  4. Start the frontend:
    cd simple-login-frontend && npm run dev
  5. Access the application:
    http://localhost:5173

üîë Default Credentials

Admin User: admin@example.com / password

Regular User: user@example.com / password
